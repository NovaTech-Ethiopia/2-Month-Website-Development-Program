# 2-Month Website Development Program

Welcome to the 2-Month Website Development Program! This program is designed to teach you the fundamentals of HTML, CSS, JavaScript, Git, and GitHub. You'll be learning and coding three days a week. By the end of this program, you will have a solid understanding of web development basics and version control with Git and GitHub, and will have created several projects to showcase your skills.

## Week 1

### Day 1: Introduction to Web Development
- Overview of web development
- Understanding the web: Clients and servers
- Setting up the development environment

### Day 2: Introduction to HTML
- Basic structure of an HTML document
- Common tags: `<html>`, `<head>`, `<body>`, `<title>`, `<h1>` to `<h6>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<ol>`, `<li>`
- Creating a simple HTML page

### Day 3: HTML Continued
- Forms: `<form>`, `<input>`, `<textarea>`, `<button>`, `<select>`, `<option>`
- Semantic HTML: `<header>`, `<nav>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- Basic HTML attributes

## Week 2

### Day 1: Introduction to CSS
- What is CSS and how it works
- Inline, internal, and external CSS
- Basic selectors: type, class, and ID selectors

### Day 2: CSS Continued
- Box model: margin, border, padding, content
- Styling text: font-family, font-size, color, text-align
- Styling links and lists

### Day 3: Advanced CSS
- Layout techniques: Flexbox
- CSS positioning: static, relative, absolute, fixed, sticky
- CSS units: px, em, rem, %, vw, vh

## Week 3

### Day 1: CSS Advanced Techniques
- CSS Grid layout
- Responsive design: Media queries
- CSS transitions and animations

### Day 2: Introduction to JavaScript
- Basics of JavaScript: Variables, data types, operators
- Adding JavaScript to a webpage: `<script>` tag
- Basic DOM manipulation: `document.getElementById`, `document.querySelector`

### Day 3: JavaScript Fundamentals
- Functions: Declaration, invocation, parameters, return values
- Conditional statements: `if`, `else if`, `else`
- Basic event handling: `onclick`, `onchange`

## Week 4

### Day 1: JavaScript Basics
- Arrays: Creating, accessing, modifying
- Objects: Creating, accessing properties
- Simple loops: `for`, `while`

### Day 2: JavaScript and HTML
- Changing content: `innerHTML`, `textContent`
- Changing styles: `style` property
- Basic form validation

### Day 3: Practical JavaScript
- Creating and appending elements dynamically
- Handling user input: Event listeners
- Building a simple interactive webpage (e.g., a color changer or simple calculator)

## Week 5

### Day 1: JavaScript Project
- Planning a small project: To-Do List application
- Setting up HTML structure
- Implementing CSS styling

### Day 2: JavaScript Project Continued
- Adding JavaScript to manage tasks (add, remove, mark as done)
- Storing tasks in an array
- Displaying tasks dynamically

### Day 3: JavaScript Project Completion
- Finalizing functionality
- Debugging and testing
- Styling and improving user experience

## Week 6

### Day 1: CSS Practice
- Review basic CSS concepts
- Apply CSS to a simple webpage
- Experiment with different CSS properties

### Day 2: JavaScript Practice
- Review JavaScript basics
- Build a small interactive project (e.g., a quiz or image gallery)
- Practice DOM manipulation and event handling

### Day 3: Introduction to Git and GitHub
- What is version control?
- Introduction to Git: Installing Git, basic commands (`init`, `add`, `commit`)
- Setting up a GitHub account

## Week 7

### Day 1: Git Basics
- Branching and merging
- Working with remote repositories: `clone`, `push`, `pull`
- Resolving merge conflicts

### Day 2: Collaborating on GitHub
- Forking and cloning repositories
- Creating pull requests
- Reviewing pull requests

### Day 3: Project Planning
- Brainstorm project ideas
- Define project requirements and features
- Create wireframes and mockups

## Week 8

### Day 1: Project Development
- Set up project folders and files
- Build HTML structure
- Implement basic CSS styling

### Day 2: Project Development Continued
- Add JavaScript functionality
- Test and debug code
- Refine design and layout

### Day 3: Project Refinement
- Test project across different devices and browsers
- Make necessary adjustments based on feedback
- Finalize project documentation

### Final Day: Presentation and Review
- Present final project to peers or mentors
- Receive feedback and suggestions
- Reflect on the learning journey and next steps

---

### Project Idea:
- Build a personal portfolio website showcasing your skills, projects, and experiences. Include sections for a brief introduction, projects gallery, resume/CV, and contact information. Make it visually appealing and responsive.
